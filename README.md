# 🌐 Rithika's Portfolio

A modern, responsive, and interactive portfolio website showcasing my projects, technical skills, experience, and achievements as a Computer Science Engineering student passionate about AI, Machine Learning, and Full-Stack Development.

🔗 **Live Demo:** https://portfolio-rithika-mu.vercel.app/

---

## ✨ Features

- Modern responsive UI
- Smooth animations and transitions
- Interactive project showcase
- Experience & education timeline
- Technical skills section
- Downloadable resume
- Contact section
- Optimized for desktop, tablet, and mobile devices

## 💻 Getting Started

Clone the repository

```bash
git clone https://github.com/rithikagirish/portfolio-rithika.git
```

Navigate to the project

```bash
cd portfolio-rithika
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```
